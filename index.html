Informacion General del sitio web

En este Objeto virtual de información encontraremos, contenido relacionado al Diseño Web, tenemos un menú horizontal donde nos muestra lo que llevara nuestra página web como lo es una página de inicio, en la página de lecturas donde añadimos un submenú desplegable con lecturas para ayudar a alimentar el conocimiento y actividades para reforzar lo aprendido, en la página de multimedia encontraremos videos que nos ayudaran aprender de manera más sencilla y explicada lo que se debe saber sobre el Diseño Web en HTML y como editar las funciones con CSS, en el botón de actividades nos muestra actividades en línea donde podemos afianzar conocimientos relacionados con el área a trabajar, en la página de autores vamos a encontrar los estudiantes que participamos en la creación de esta página web. Este Ovi está dirigido a todas aquellas personas que quieran incursionar en el mundo del desarrollo Web de una manera básica ya que diseñar paginas es un tema muy extenso pero muy interesante.

Objetivos del OVI

Incentivar a los estudiantes a tomar el curso.

Mostrar los conocmientos abquiridos hasta ahora.

Desarrollar una página Web.

Que el estudiante se familiarice con el código.

Objetivos del Curso

Se espera que al finalizar el curso, el estudiantes contruya sitios web, por medio de un sistema de control de versiones con lenguajes de marca en HTML5 y CSS3, para potenciar las habilidades en el desarrollo de metodologías y herramientas especializadas en linea.

Aprender que es y para que sirven HTML y CSS.

Diseñar y Desarrollar una Página Web funcional.
