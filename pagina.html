Informacion General del sitio web

En este Objeto virtual de información encontraremos, contenido relacionado al Diseño Web, tenemos lecturas para ayudar a alimentar el conocimiento y actividades para reforzar lo aprendido, abra videos de ayuda para refrescar y aprender de manera más sencilla lo que se debe saber sobre el Diseño Web. Este Ovi está dirigido a todas aquellas personas que quieran incursionar en el mundo del desarrollo Web

Objetivos del OVI

Incentivar a los estudiantes a tomar el curso. Mostrar los conocmientos abquiridos hasta ahora. desarrollar una página Web. Que el estudiante se familiarice con el código.

Objetivos del curso

Se espera que al finalizar el curso, el estudiantes contruya sitios web, por medio de un sistema de control de versiones con lenguajes de marca en HTML5 y CSS3, para potenciar las habilidades en el desarrollo de metodologías y herramientas especializadas en linea. Aprender que es y para que sirven HTML y CSS. Diseñar y Desarrollar una Página Web funcional.
